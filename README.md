# TwitterStreaming

## Setting up 

Clone repo:\
```git clone git@github.com:BrianCoveney/TwitterStreaming.git```

Go to project's directory:\
```cd TwitterStreaming```

Get required dependencies:\
```go get ./...```

Launch application:\
```docker-compose up```

App frontend V1:\
```http://localhost:3000/sentiment```

App frontend V2:\
```http://localhost:9090/sentiment```
